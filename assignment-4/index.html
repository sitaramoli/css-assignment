One of the method to find whether an element is block or inline is to use an element cosecutively for two times. If second element starts in a new line then the element is block. Element is inline if second element starts in a same line.

Block elements

Block elements take the full width of their parent container and create new line after them. We can set the height and width of block elements. Examples of block elements are:

<p>

This is paragraph 1

This is paragraph 2

<div>

This is div 1
This is div 2

<h1-h6>

This is header 2

This is header 4

<ol>,<ul>,<li>

  1. Item 1
  2. Item 2
  • Item 1
  • Item 2

<hr> - creates a horizontal line

<table>

S.N Item
1 Tea
S.N Name
1 Coffee

<header>

This is header 1
This is header 2

<footer>

This is footer 1
This is footer 2

Inline elements

Inline elements only occupy the with they require. We can't set the with and height of inline elements. Examples of inline elements are:

<a>

Link 1 Link 2

<span>

Span 1 Span 2

<textarea>

<select>,<option>

<b>

Text 1 Text 2

<input>

<button>

<img>

<i>

Text 1 Text 2

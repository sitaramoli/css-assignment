Block elements

Block elements take the full width of their parent container and create new line after them. We can set the height and width of block elements. Examples of block elements are:

  1. <p> - paragraph
  2. <div> - a container for contents
  3. <h1-h6> - headings
  4. <ol> - ordered list
  5. <ul> - unordered list
  6. <li> - list item
  7. <hr> - creates horizontal line
  8. <table> - a table
  9. <header> - a header
  10. <footer> - a footer

Inline elements

Inline elements only occupy the with they require. We can't set the with and height of inline elements. Examples of inline elements are:

  1. <a> - a hyperlink which is used to link pages
  2. <span> - a generic inline container
  3. <textarea> - multiline text input field
  4. <select> - a dropdown menu
  5. <option> - defines an option that can be selected in the dropdown menu
  6. <b> - bold text
  7. <input> - form input field
  8. <button> - a button
  9. <img> - image
  10. <i> - italic text

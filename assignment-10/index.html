min()

It allows us to set the smallest value from a list of comma-separated expression as the value of CSS property value.

.min{
width: min(400px,50%);
}
Width of this div is selected from the min() function. Among 400px and 50% of containing element's width,the smallest value is selected as the width of div.

max()

max() will set the largest (most positive) value from a list of comma-separated expressions as the value of a CSS property value.

.max{
width: max(400px,50%);
}
Width of this div is selected from the max() function. Among 400px and 50% of containing element's width,the largest value is selected as the width of div.

clamp()

It clamps a middle value within a range of values between a defined minimum bound and a maximum bound. The function takes three parameters: a minimum value, a preferred value, and a maximum allowed value.

.clamp{
width: clamp(200px,60%,700px);
}
Preferred width of this div is 60% of viewport's width. If 60% of viewport's width is less than 200px, then 200px is used as width of div. If 60% of viewport's width is greater than 700px, then 700px is used as width of div.

Media queries allow us to apply different styles to an element based on the characteristics of the device or viewport. In addition to that, min(), max(), and clamp() allow us to apply different styles to an element based on the characteristics of the containing element.

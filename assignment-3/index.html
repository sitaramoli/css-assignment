box-sizing property

`box-sizing` property allows us to control the size of element's content box, padding box, and border box.

By default, the size of element's content box is determined by the width and height properties. The padding and border are then added outside of the content box to determine the size of the element's box.

With `box-sizing` property we can change change this behavior and have the padding and border included in the size of content box.This is useful when we want to set the width and height of an element and have the padding and border fit within those dimensions.

Following example shows the use of `box-sizing` property to create an alternate box model:
.border-element{
width: 200px;
height: 200px;
padding: 20px;
border: 2px solid black;
box-sizing: border-box;
}

This is an element with the given properties.

Here `box-sizing` property is set to `border-box`, so the padding and border will be included in the size of content box.The total width and heiht of element will be 200px and 200px respectively.

This example shows the use of `box-sizing: content-box` property:
.padding-element{
width: 200px;
height: 200px;
padding: 20px;
border: 2px solid black;
box-sizing: content-box;
}

This is an element with the given properties.

Here `box-sizing` property is set to `content-box`. The total width of an element will be 244px(200px + 20px padding on each side + 2px border on each side), and total height will be 244px(200px + 20px padding on each side+ 2px border on each side).

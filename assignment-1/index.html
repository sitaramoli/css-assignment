Examples of state selectors are:

:hover

- selects an element when user hovers a mouse over it.

:active

-selects an element when it is being activated.

:checked

- selects a checked radio button or checkbox.

:focus

-selects an element that has focus.

:disabled

selects a disabled element

:invalid

selects any form,fieldset,input or other form elements that fail to validate.

Some level 4 selectors

:blank

selects empty user input elements like <input>,<textarea>

:user-invalid

selects user-altered user-input elements with invalid input.

:nth-col(n)

selects element that represents a cell belonging to the nth column in a grid/table
